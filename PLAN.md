# 大学社交平台 — 项目计划 (v2)

> 蓝图来源：`~/sucai_0/06.txt` — 三期功能蓝图

## 技术栈

前端 uni-app 3.x + Vue 3 + TS + Pinia + uView Pro
后端 Go + Gin + PostgreSQL + Redis
管理后台 Vue 3 + Element Plus
部署 Docker + Compose
