# Project: university-social (校园社)

## Session Start (MANDATORY)
1. Read `.opencode/PROJECT_STATUS.md`
2. `elf search query="university-social"` load relevant past learnings
3. `opencode-mem search semantic query="university social app"` load persistent memory

## Session End (MANDATORY)
1. Update `.opencode/PROJECT_STATUS.md`:
   - `overall` progress %
   - `last_updated` to today
   - Add completed items to `## 完成功能`
   - Move completed from `## 待办功能`
   - Update `## 已知问题`
   - Add new `## 技术决策`
2. `elf add-rule type=learning content="<session-summary>"` save session learnings
